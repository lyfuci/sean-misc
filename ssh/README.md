# 简述

Secure Shell (SSH) 是一种加密网络协议，用于在不安全的网络上安全地运行网络服务。典型应用包括远程命令行、登录和远程命令执行，但任何网络服务都可以通过 SSH 进行保护。

常见用法:
* 登录及文件传输
* 隧道(端口转发)


# 用法示例
## 登录及文件传输
这可能是 ssh 最常用的用法了，使用 ssh 登录仅需使用客户端连接到指定服务器的 IP 端口即可, 在此就不赘述了。文件传输的用法可以参见 sftp 和 scp 相关协议。
